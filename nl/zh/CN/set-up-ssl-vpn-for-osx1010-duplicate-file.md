# 针对 Mac OS X 10.10 设置 SSL VPN

针对 MacOS StandAlone Array SSL VPN Motion Pro Plus 客户机的步骤：

1. 从 Apple 商店安装“Motion Pro Plus”客户机。

2. 在“应用程序”文件夹下找到 Motion Pro Plus 客户机，然后打开该应用程序。

3. 单击“概要文件”，然后单击“添加”。

4. 填写“站点名称”、“主机”和“用户名”，然后单击“确定”。

5. 单击左上方的 VPN，然后进行连接。
